Разработайте приложение, позволяющее рисовать при помощи мыши на растровом изображении. Изображение пользователь может либо создать с нуля (File->New), либо выбрать существующий файл изображения (File->Open). Программа должна поддерживать хотя бы один из стандартных форматов изображений: PNG, JPEG, BMP. Рисование начинается при нажатии левой кнопки мыши внутри клиентской области и продолжается при перемещении мыши при нажатой кнопке мыши аналогично инструментам «Кисть» или «Карандаш» в графическом редакторе Paint. Толщину рисуемых линий выберите на своё усмотрение.
Пункт меню File->Save As позволяет пользователю сохранить рисунок в одном из следующих форматов: PNG, BMP, JPEG.

Бонус - 20 баллов
Бонус начисляется за возможность выбора цвета рисования. Используйте либо стандартный диалог выбора цвета, либо разработайте собственный.
